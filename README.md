# multiETM
This is code that accompanies the paper titled "multiETM: Multimodal Embedded Topic Model for Electronic Health Records". <br>  

## Dependencies <br>  
* Python 3.6.12 <br>  
* Pytorch 1.8.1 <br>  

## Result <br>  
 Model     | TC    | TD    | Interpretability    | Predictive Loss   
 -------- | :-----------:  | :-----------:  | :-----------: | :-----------: 
 multiETM    | 0.130     | 0.888    | 0.115      | 113.6
